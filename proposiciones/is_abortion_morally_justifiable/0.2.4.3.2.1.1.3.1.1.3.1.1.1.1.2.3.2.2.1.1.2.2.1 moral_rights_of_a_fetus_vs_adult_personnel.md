## conclusion

The argument collapses into a recursive loop because 'person' is used to define 'rights' which is used to define 'person' without a grounding external to the host's biolgy. Using first principles, a moral agent (adult) requires higher-order neural integration which a 1-celled zygote lacks, creating a structural contradiction in the 'equivalence' claim, and for all this the assertion that a fetus has identical moral rights to an adult is an uncollapsable subjective axiom rather than a logical necessity.
