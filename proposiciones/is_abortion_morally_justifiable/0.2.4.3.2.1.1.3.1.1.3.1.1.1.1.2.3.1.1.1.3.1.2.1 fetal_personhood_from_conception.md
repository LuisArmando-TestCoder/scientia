## logical_tension

- **paradox**: The Potentiality Paradox: Treating a potential x as a current x creates a logic where every sperm or egg cell is a 'potential' person, yet we only count the 'fused' ones.
- **ambiguity**: The word 'human' is used both as a biological descriptor (species) and a moral one (person).
- **useful_contradiction**: A fetus is both 'part of the mother' (biologically dependent) and 'distinct' (genetically unique).

- If conception is not the threshold, the system must define a new, equally 'non-arbitrary' point (birth, viability, heartbeat).
